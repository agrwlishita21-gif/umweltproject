# Getting Started

Welcome to Umwelt, a tool for creating accessible multimodal data representations. With Umwelt, you can transform your data into coordinated visualizations, sonifications, and accessible text descriptions.

## What is Umwelt?

Umwelt is designed primarily for screen reader users, enabling you to create multiple ways to experience the same data:

- **Text**: Rich, hierarchical descriptions that work seamlessly with screen readers
- **Audio**: Data sonification that maps data to sound properties like pitch and volume
- **Visual**: Charts and plots that coordinate with the other modalities

All three modalities are generated from a single specification, ensuring consistency across representations. While built for screen reader users first, Umwelt is useful for anyone who wants to explore data through multiple senses.

## Quick Start

1. **Upload your data** - Start with a CSV file or choose from example datasets
2. **Configure fields** - Set data types and select which fields to include
3. **Design visualizations** - Create charts by mapping fields to visual properties
4. **Design sonifications** - Map fields to audio properties like pitch and volume
5. **Experience your data** - View, listen, and read your multimodal representation

## Interface Overview

Umwelt has two main panels:

### Editor (Left Panel)
The editor has four tabs:
- **Data**: Upload files and manage datasets
- **Fields**: Configure field types and properties
- **Visual**: Design charts and visualizations
- **Audio**: Design sonifications and audio mappings

### Viewer (Right Panel)
The viewer shows your results in real-time:
- **Visualization**: Rendered charts
- **Description**: Accessible text descriptions
- **Sonification**: Audio playback controls
